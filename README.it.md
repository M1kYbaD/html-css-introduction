## Cosa Imparerai

### Fondamenti di HTML

- Comprendere la struttura base di un **documento HTML** tramite i `tag`.
- Utilizzare gli **elementi HTML principali** come `titoli`, `paragrafi`, `immagini`, `link`, `liste` e `sezioni`.
- Scrivere codice **semantico** e **accessibile**.

### Basi di CSS

- Applicare **CSS** per stilizzare `testo`, `layout`, `colori` e `spaziature`.
- Imparare i **selettori CSS** pi√π comuni e come **selezionare gli elementi**.
- Utilizzare tecniche di layout come `flexbox` e il `posizionamento` per strutturare i contenuti.

### Design di Carousel di Immagini e Contenuti

- Esplorare come i **carousel** sono strutturati in **HTML** e stilizzati con **CSS**.
- Imparare a implementare `slide` e `controlli di navigazione`.
- Fare pratica combinando **transizioni** e tecniche di layout per creare **componenti UI interattivi**.

## Contenuto

- `Simple-Exercises`: una serie di piccoli esercizi per fare pratica con le **basi di HTML e CSS**.
